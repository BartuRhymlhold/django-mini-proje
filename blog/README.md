# Django Mini Blog
Bu, bir Django mini blog uygulamasıdır.

Basit bir blog uygulaması. Kullanıcılar yazı ekleyebilir, silebilir, güncelleyebilir.

## Özellikler
- Kayıt / Giriş
- Yazı ekleme
- Yazı silme / güncelleme
- Profil güncelleme
- Admin panel

## Kurulum
1. `git clone`
2. `python manage.py migrate`
3. `python manage.py runserver`
